# Фильтрация и сортировка товаров в интернет-магазине

## Описание проекта
В рамках домашнего задания реализован функционал каталога интернет-магазина на Django.

Пользователь может:
- просматривать список товаров;
- фильтровать товары по категории;
- фильтровать товары по диапазону цен;
- фильтровать товары по дате добавления;
- сортировать товары по популярности;
- сортировать товары по цене (по возрастанию и убыванию);
- сортировать товары по дате добавления;
- переходить между страницами каталога с пагинацией по 10 товаров;
- обновлять список товаров без перезагрузки страницы с помощью AJAX.

## Что реализовано
- модель `Product` с полями:
  - `name`
  - `category`
  - `price`
  - `created_at`
- модель `Order`
- модель `OrderItem`
- фильтрация товаров по категории, цене и дате добавления;
- сортировка по популярности, цене и дате;
- пагинация;
- AJAX-подгрузка данных через `fetch()`;
- использование Django ORM:
  - `filter()`
  - `annotate()`
  - `Count()`
  - `Q()`
  - `order_by()`

## Использованные технологии
- Python
- Django
- SQLite
- HTML
- JavaScript
- AJAX (`fetch`)

1. Перейти в папку проекта:
   ```bash

2.Установить зависимости, если нужно:

python manage.py makemigrations
python manage.py migrate
